# 算法服务架构文档

本算法服务 (`services/algorithm`) 采用“两阶段检测”创新架构：利用目标检测与聚类算法初步感知，再结合多模态大语言模型（Qwen-VL）进行深入语义分析。

## 算法 Pipeline

```text
RTSP视频流 ──► 帧提取 ──► YOLO交通对象检测 ──► 群组分析 ──► 交通道路检测 ──► 结果输出
                               │                │               │
                               ▼                ▼               ▼
                          目标边界框      群组聚类算法      Qwen-VL模型分析
```
