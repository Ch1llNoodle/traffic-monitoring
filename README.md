# 智慧交通监控预警系统

基于 YOLO 和多模态大语言模型 (LLM) 的智能交通监控预警解决方案，实现对道路交通状况的实时感知、智能分析与自动预警。

## 架构概览

本系统采用微服务架构设计，包含以下核心模块：

```text
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   前端服务       │◄──►│   后端服务       │◄──►│   算法服务       │
│   (Vue3)        │    │  (Spring Boot)  │    │    (Flask)      │
└─────────────────┘    └─────────────────┘    └─────────────────┘
         ▲                       ▲                       ▲
         ▼                       ▼                       ▼
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   交互界面       │    │   数据存储       │    │   文件存储       │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

## 技术栈

### 核心栈
*   **前端**: Vue3, TypeScript, Vite, Element Plus
*   **后端**: Java 21, Spring Boot 3, MyBatis Plus, MySQL 8
*   **算法**: Python, Flask, PyTorch, YOLOv8/11/12, Qwen-VL (Multimodal LLM)
